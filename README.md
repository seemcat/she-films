# she films society

A static website for she films society - for girlies in their storytelling era.

## Pages

- **Home** (`index.html`) - Landing page with hero section
- **About** (`about.html`) - Origin story and beliefs
- **Sunday Ritual** (`sunday-ritual.html`) - Weekly program details

## Local Development

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Deploying to Vercel

1. Install Vercel CLI (if not already installed):
   ```bash
   npm i -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

3. Follow the prompts to link to your Vercel account and deploy

Alternatively, you can deploy via the Vercel dashboard by:
1. Go to [vercel.com](https://vercel.com)
2. Click "Add New Project"
3. Import this repository
4. Click "Deploy"

## Customization

- Replace placeholder images in the HTML files with your actual images
- Update the navigation links if you add more pages
- Modify `styles.css` to adjust colors, fonts, and layouts
